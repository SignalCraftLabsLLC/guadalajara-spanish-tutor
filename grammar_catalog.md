## 1. SER vs. ESTAR

Two verbs meaning "to be" — used in entirely different situations.

**SER** — permanent or inherent qualities:
- Identity, nationality, profession, origin, material, time, ownership
- *Es médico. / Es de México. / El anillo es de oro. / Son las tres.*

**ESTAR** — temporary states, location, ongoing action:
- Emotions, health, location, progressive tenses
- *Está cansado. / Está en Madrid. / Estoy aprendiendo español.*

**Key distinction with adjectives:**
- *Es tranquilo* (he's a calm person) vs. *Está nervioso* (he's nervous right now)
- *Es abierto* (he's an open person) vs. *Está abierto* (it's open — door/store)

## 2. HAY (There is / There are)

**HAY** — existence, availability (invariable form — no singular/plural change):
- *Hay un banco cerca.* / *Hay tres restaurantes en la calle.*
- *¿Hay leche en la nevera?*

**HAY vs. ESTÁ/N:**
- HAY introduces something for the first time (indefinite)
- ESTÁ/N locates something already known (definite)
- *Hay una farmacia al lado.* (it exists somewhere) → *La farmacia está en la esquina.* (that specific one)

**Related worksheets:** IMG_2557, IMG_2558

## 3. Verbos Regulares — Presente de Indicativo

**-AR endings:** yo -o / tú -as / él -a / nosotros -amos / ellos -an
**-ER endings:** yo -o / tú -es / él -e / nosotros -emos / ellos -en
**-IR endings:** yo -o / tú -es / él -e / nosotros -imos / ellos -en

*Caminar, comer, vivir* as model verbs.

## 4. Adverbios de Frecuencia

Used with present tense to describe habitual actions:

siempre / casi siempre / frecuentemente / normalmente / a veces / raramente / casi nunca / nunca

*Siempre como a las dos. / Nunca llego tarde.*

---

## 5. Reglas de Acentuación

**Grave (llana):** stress on penultimate syllable — words ending in vowel, -n, -s (no written accent needed)
**Agudo (aguda):** stress on last syllable — words ending in consonants except -n, -s (no written accent needed)
**Esdrújula (antepenúltima):** stress on third-to-last syllable — ALWAYS written accent: *cámara, teléfono, médico*

Exception: when stress doesn't follow the rule → written accent always required.

Interrogatives always carry accent: *¿qué / quién / cuándo / dónde / cuánto / cuál / cómo / por qué*

## 6. Verbos Irregulares — Solo Forma YO

| Verb | Yo |
|------|-----|
| hacer | hago |
| dar | doy |
| saber | sé |
| salir | salgo |
| caer | caigo |
| caber | quepo |
| oír | oigo |
| valer | valgo |
| traer | traigo |

---

## 7. Verbos Irregulares — Patrón -cer / -cir / -uir → -zco

Verbs ending in a vowel + -cer or -cir: yo form changes to -zco:
- *conocer → conozco / parecer → parezco / traducir → traduzco*

Verbs ending in -uir (not -guir): insert -y- in all forms except nosotros/vosotros:
- *construir → construyo, construyes, construye, construimos, construyen*

---

## 8. Verbos Irregulares con Cambio Vocálico (Presente)

The stem vowel changes in all persons EXCEPT nosotros (and vosotros):

| Change | Examples |
|--------|----------|
| o > ue | poder, dormir, volver, recordar, encontrar, soler, costar |
| e > ie | querer, pensar, cerrar, entender, preferir, sentir, empezar |
| e > i | pedir, servir, seguir, repetir, medir, elegir, reír |
| u > ue | jugar (only common verb) |

*PODER:* puedo / puedes / puede / podemos / pueden
*PEDIR:* pido / pides / pide / pedimos / piden
*JUGAR:* juego / juegas / juega / jugamos / juegan

## 10. Descripción Física (SER + TENER)

**SER** for physical characteristics:
- *Es alto/bajo/delgado/gordo/rubio/moreno/pelirrojo*
- *Es guapo/feo/joven/viejo*

**TENER** for features:
- *Tiene los ojos azules. / Tiene el pelo largo y rizado.*
- *Tiene barba. / Tiene pecas.*

| Español | Inglés |
|---------|--------|
| ¿Qué? | What? |
| ¿Quién? / ¿Quiénes? | Who? |
| ¿Cuándo? | When? |

| ¿Cuál? / ¿Cuáles? | Which? |
| ¿Cómo? | How? |
| ¿Por qué? | Why? |
| Porque | Because (not interrogative — the answer) |

Always written with accent mark. Always preceded by ¿ in Spanish.

## 12. Concordancia Sustantivo-Adjetivo (Gender + Number Agreement)

Adjectives must agree in gender AND number with the noun they modify:

- *un chico alto / una chica alta*
- *unos chicos altos / unas chicas altas*

Adjectives ending in -e or consonant: only change for number:
- *un hombre inteligente / una mujer inteligente / personas inteligentes*

---

## 13. Artículos

**Definite (the):** el / la / los / las
**Indefinite (a/an/some):** un / una / unos / unas

Special: use *el* before feminine nouns starting with stressed *a-*:
- *el agua / el águila* (but: *las aguas*)

## 18. GUSTAR + Verbos Similares (Reverse-subject structure)

Structure: (A mí/ti/él...) + me/te/le/nos/os/les + **GUSTA** (singular/infinitive) / **GUSTAN** (plural)

*A mí me gusta el café. / Me gustan los tacos.*

**Other verbs with the same structure:**
encantar / interesar / doler / apetecer / sorprender / parecer / molestar / importar / faltar

*Me duele la cabeza. / Nos interesa el español. / ¿Te apetece un café?*

## 19. Pronombres de Objeto Indirecto (POI)

| Person | Pronoun |
|--------|---------|
| yo | me |
| tú | te |
| él/ella/usted | le |
| nosotros | nos |
| ellos/ustedes | les |

Placed BEFORE the conjugated verb (or attached to infinitive/gerund):
*Juan le compró flores (a su mamá). / Me mandó un mensaje.*

## 20. Verbos Reflexivos

The subject performs AND receives the action. Reflexive pronouns:
*me / te / se / nos / os / se*

Position: before conjugated verb or attached to infinitive/gerund:
*Me levanto a las ocho. / Voy a bañarme. / Estoy vistiéndome.*

Common reflexive verbs: levantarse, bañarse, ducharse, vestirse, despertarse, acostarse, irse, sentirse, llamarse, casarse

## 21. La Hora (Telling Time)

**SER + las/la + número:**
- *¿Qué hora es? — Son las tres. / Es la una.*
- *Son las tres y cuarto. / Son las tres y media. / Son las cuatro menos diez.*

*mediodía* (noon) / *medianoche* (midnight)
*de la mañana / de la tarde / de la noche*

---

## 22. Tiempo vs. Vez

**Tiempo** — duration/weather (uncountable sense of time):
*¿Cuánto tiempo llevas aquí? / Hace buen tiempo.*

**Vez** — a countable instance/occurrence:
*¿Cuántas veces has ido? / Una vez, dos veces, a veces.*

## 24. Comparaciones

**Más/menos + adjetivo + que:** *Es más alto que su hermano.*
**Tan + adjetivo + como:** *Es tan inteligente como ella.*
**Tanto/a/os/as + sustantivo + como:** *Tengo tantos libros como tú.*
**Tanto como** (with verbs): *Trabaja tanto como yo.*

Irregular comparatives: *bueno → mejor / malo → peor / grande → mayor / pequeño → menor*

---

## 25. Posición del Adjetivo + Apócope

Most adjectives follow the noun in Spanish: *un coche rojo / una chica alta*

Some adjectives PRECEDE the noun; some shorten before masculine singular nouns:
- *bueno → buen perro* / *malo → mal tiempo*
- *primero → primer piso* / *tercero → tercer lugar*
- *grande → gran hombre* (before any gender — different meaning: "great")

---

## 26. Bien vs. Bueno

**Bien** — adverb (modifies verbs/adjectives): *Habla bien. / Estoy bien.*
**Bueno/a** — adjective (modifies nouns): *Es un buen libro. / La comida está buena.*

Common confusion: *Estoy bien* (I'm fine) ≠ *Estoy bueno* (I'm hot/tasty — slang 🌵)

## 27. Presente Continuo (Estar + Gerundio)

**Structure:** ESTAR (present) + GERUNDIO

Gerund formation:
- -AR → -ando: *caminar → caminando*
- -ER/-IR → -iendo: *comer → comiendo / vivir → viviendo*

**Irregular gerunds:**
*leer → leyendo / oír → oyendo / caer → cayendo / construir → construyendo*
*dormir → durmiendo / decir → diciendo / venir → viniendo / ir → yendo*
*vestir(se) → vistiéndose / sentir → sintiendo / pedir → pidiendo*

Use: action happening RIGHT NOW (not habitual)
*Estoy comiendo ahora mismo. / ¿Qué estás haciendo?*

Time markers: *ahora / en este momento / ahora mismo*

## 28. Cortesía y Formalidad 🌵

**Levels of formality (from most to least formal):**

| Level | Form | Example |
|-------|------|---------|
| Muy formal | Usted condicional | ¿Me podría abrir la puerta? |
| Formal | Usted presente | ¿Me puede abrir la puerta? |
| Semi-formal | Tú condicional | ¿Me podrías abrir la puerta? |
| Informal | Tú presente | ¿Me puedes abrir la puerta? |
| Muy informal | Imperativo | Ábreme la puerta. |

**Interrupting / Excuse me:**
- *Disculpa* → informal (tú) — interrupting someone or looking for something
- *Disculpe* → formal (usted)
- *Con permiso* → I need to pass / excuse me physically
- *Perdón* → apology for an error

🌵 In Guadalajara and Mexico generally, **Usted** is used more broadly than in Spain — even with people not much older than you, in shops, and with strangers. Using *tú* with a stranger can feel abrupt.

## 29. Futuro Próximo (Ir + a + Infinitivo)

**Structure:** IR (present) + A + INFINITIVO

*Voy a comer. / Van a viajar. / ¿Vas a venir?*

Also used with *querer* + infinitivo and *pensar* + infinitivo to express plans:
*Quiero ir al mercado. / Pienso estudiar esta noche.*

**Related worksheets:** IMG_2526

## 30. Sustantivos Contables vs. No Contables + Cuantificadores

**Contables (countable):** can be pluralized — *libros, manzanas, personas*
**No contables (uncountable):** mass nouns — *agua, leche, arroz, azúcar*

**Quantifiers:**
- *mucho/a/os/as* — a lot of
- *poco/a/os/as* — a little / few
- *bastante/s* — quite a lot / enough
- *demasiado/a/os/as* — too much / too many
- *algunos/as* — some (countable)
- *ningún / ninguna* — none

---

## 31. Antes de / Después de + Infinitivo

**Antes de** + infinitivo → before doing something:
*Me cepillo los dientes antes de acostarme.*

**Después de** + infinitivo → after doing something:
*Después de comer, tomo un café.*

Note: infinitivo (not conjugated verb) always follows these prepositions.

## 32. Pretérito Simple / Indefinido

**Use:** completed actions with a defined time; single/sequential past events.
Markers: *ayer / la semana pasada / hace dos años / en 1990 / el lunes / una vez*

**Regular endings:**

|-|-----|---------|
| yo | -é | -í |
| tú | -aste | -iste |
| él | -ó | -ió |
| nosotros | -amos | -imos |
| ellos | -aron | -ieron |

**Irregular verbs (strong stems):**
| Verb | Stem | Forms (yo/tú/él/nos/ellos) |
|------|------|---------------------------|
| tener | tuv- | tuve/tuviste/tuvo/tuvimos/tuvieron |
| estar | estuv- | estuve... |
| poder | pud- | pude... |
| poner | pus- | puse... |
| saber | sup- | supe... |
| caber | cup- | cupe... |
| querer | quis- | quise... |
| hacer | hic- | hice/hiciste/**hizo**/hicimos/hicieron |
| venir | vin- | vine... |

## 33. Pretérito Imperfecto

**Use:** habitual past actions; background description; ongoing/interrupted past states; no defined end point.
Markers: *antes / siempre / frecuentemente / normalmente / cuando era niño / de niño*

**Regular endings:**
| | -AR | -ER/-IR |
|-|-----|---------|
| yo | -aba | -ía |
| tú | -abas | -ías |
| él | -aba | -ía |
| nosotros | -ábamos | -íamos |
| ellos | -aban | -ían |

**Irregular verbs (only 3!):**
| | SER | IR | VER |
|-|-----|----|----|
| yo | era | iba | veía |
| tú | eras | ibas | veías |
| él | era | iba | veía |
| nosotros | éramos | íbamos | veíamos |
| ellos | eran | iban | veían |

**Four uses:**
1. **Habitual past:** *Iba al cine todos los domingos.*
2. **Background description:** *La casa era grande y tenía jardín.*
3. **Interrupted action:** *Caminaba por la calle cuando me caí.* (imperfecto = background; indefinido = interruption)
4. **Simultaneous ongoing actions:** *Mientras leías, yo cocinaba.*

## 34. Pretérito Perfecto

**Use:** past experiences (ever/never); actions started in past and continuing now; recent past (still feels relevant).

**Structure:** HABER (presente) + PARTICIPIO PASADO

| HABER | |
|-------|--|
| yo he | nosotros hemos |
| tú has | vosotros habéis |
| él/ella ha | ellos han |

**Regular participles:**
- -AR → -ado: *caminar → caminado*
- -ER/-IR → -ido: *comer → comido / vivir → vivido*

**Irregular participles (-to form):**
ver → visto / abrir → abierto / escribir → escrito / volver → vuelto
poner → puesto / romper → roto / morir → muerto / cubrir → cubierto
descubrir → descubierto / devolver → devuelto / describir → descrito

**Irregular participles (-cho/-so):**
decir → dicho / hacer → hecho / deshacer → deshecho / imprimir → impreso

**Examples:**
- *¿Alguna vez has comido tacos de canasta?* (experience)
- *Hemos vivido aquí por diez años.* (ongoing from past)
- *He comido hace una hora.* (recent past)

## 36. Objeto Directo (OD) + Pronombres

The direct object answers *¿Qué?* or *¿A quién?* after the verb.

**OD Pronouns:**
| Person | Pronoun |
|--------|---------|
| yo | me |
| tú | te |
| él/ello | lo |
| ella | la |
| nosotros | nos |
| ellos | los |
| ellas | las |

*El perro come la carne. → El perro la come.*
*Compré los libros. → Los compré.*

Position: before the conjugated verb, or attached to infinitive/gerund.

## 37. Objeto Indirecto (OI) + Pronombres + Pronombres Dobles

**OI Pronouns:** me / te / **le** / nos / **les**

*Juan le compró flores (a su mamá). → Juan le compró flores.*

**Double object pronouns (OI + OD together):**
When both appear, OI comes first: **OI + OD + verb**
le/les → **se** when followed by lo/la/los/las

*Juan le compró flores a su mamá. → Juan se las compró.*
*Mi novia me trajo chocolates. → Mi novia me los trajo.*

## Resumen: Las Tres Formas del Pasado

| Tense | Use | Marker words |
|-------|-----|-------------|
| **Pretérito Simple/Indefinido** | Completed, specific, sequential | ayer, hace X años, en 2005, de repente |
| **Pretérito Imperfecto** | Habitual, background, ongoing | siempre, antes, mientras, cuando era niño |
| **Pretérito Perfecto** | Experience, recent, still relevant | alguna vez, nunca, hoy, ya, recientemente |

**The classic contrast:**
*Cuando era niño, vivía en Kansas. Un día, mi familia se mudó a Texas.*
→ [imperfecto: background] + [indefinido: the event that changed it]
